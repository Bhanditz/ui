To get the help system up and running:

1. Clone "new" branch from https://github.com/JetBrains/ui.git repository.

   git clone https://github.com/JetBrains/ui.git -b new


2. Adjust config.properties file:

    primary.source.folder=<the folder in which you clone help project sources> (from #1).


3. Run the help-builder.jar:

    java -Dhelp.builder.config.file=config.properties -jar help-builder.jar 

Open http://127.0.0.1:9993/help/ui-guidelines/.
